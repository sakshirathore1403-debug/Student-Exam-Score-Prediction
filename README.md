This project is a Machine Learning model that predicts student exam scores based on the number of hours studied. The model is developed using Python and the Linear Regression algorithm from the Scikit-learn library.
<br>
The goal of this project is to demonstrate the basic workflow of a Machine Learning project including data preprocessing, model training, evaluation, and prediction.
<br>
<ul><h3>Features of this model</h3>
  <li>Data preprocessing and exploration using pandas</li><br>
  <li>Visualization of data using Matplotlib</li><br>
  <li>Training a Linear Regression model using Scikit-learn</li><br>
  <li>Model evaluation using Mean Absolute Error and Mean Squared Error</li><br>
  <li>Predicting student scores based on given data</li><br>
</ul>
<ul><h3>Tools and Technologies</h3>
  <l>Python</l><br>
  <l>Pandas</l><br>
  <l>NumPy</l><br>
  <l>Matplotlib</l><br>
  <l>Scikit-learn</l><br>
  <l>Streamlit (web interface)</l><br>
</ul>
<ul><h3>Project Workflow</h3>
  <l>Load and explore the dataset</l><br>
  <l>Perform data preprocessing</l><br>
  <l>Split the dataset into training and testing sets</l><br>
  <l>Train the Linear Regression model</l><br>
  <l>Evaluate model performance</l><br>
  <l>Make predictions on new data</l><br>
</ul>

<ul><h3>Run the Project</h3>
  <li>Clone the repository : https://github.com/sakshirathore1403-debug/Student-Exam-Score-Prediction.git</li><br>
  <li>Navigate to the project folder : cd score-prediction-model (path of the folder)</li><br>

</ul>
